## 🏗️ Selected Projects

### 🎬 CineStream – Movie Streaming Platform
**[Repo Link](https://github.com/TunDuzz/cinestream)** 

**Responsibilities:**
- Developed **15+ RESTful APIs** for managing movies, actors, authentication, and comments using **ASP.NET Core**.
- Integrated external movie data using the **TMDB API**, managing a database of **1000+ records**.
- Implemented **JWT-based authentication** and **role-based authorization (RBAC)** to secure endpoints.
- Applied **Clean Architecture** to structure the project into Domain, Application, Infrastructure, and API layers.
- Optimized performance using **in-memory caching (MemoryCache)** and implemented **pagination & filtering**.
- Added **Rate Limiting** middleware to control request flow and prevent API abuse.
- Containerized the application using **Docker Compose** and configured **Nginx** reverse proxy with **SSL (Let's Encrypt)**.
- Documented and tested all APIs using **Swagger/OpenAPI**.

**Tech Stack:** ASP.NET Core, EF Core, SQL Server, MemoryCache, Docker, Nginx, JWT.
